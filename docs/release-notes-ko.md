**PDF Redactor M**은 PDF 문서의 개인정보를 안전하게 보호하는 모바일 마스킹 도구입니다. 계약서, 신분증 등 민감 정보를 선택적으로 가리고, 자동 탐지 기능으로 전화번호·주민번호·이메일 등을 빠르게 찾아냅니다.

## 주요 기능
**📄 PDF 편집**
- 손가락 드래그로 간편한 영역 선택 및 영구 마스킹
- 전화번호, 주민번호, 이메일 자동 감지
- 페이지 이동, 확대/축소, 목차 탐색

**🎨 커스터마이징**
- 흰색/검은색 마스킹 색상 선택
- 스포이드로 PDF에서 색상 추출
- 실시간 확대경 프리뷰

## 🎯 추천 대상

- 📋 직장인: 계약서·제안서 민감 정보 가리기
- 🏥 의료/법률: 환자·의뢰인 정보 보호
- 🎓 학생/연구자: 논문 개인정보 제거
- 👤 개인: 신분증·통장 사본 안전 제출
